## 🛠️ Installation Steps

1. Clone the repository

```bash
git clone https://github.com/guslizikri/tickitz-api
```

2. Install dependencies

```bash
npm install
# or
yarn install
```

3. Run the app

```bash
npm start
```

🌟 You are all set!
