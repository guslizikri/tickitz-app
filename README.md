## About The Project

 <p>
 <strong>Tickitz App</strong> Merupakan  website pembelian tiket yang dapat memudahkan pembelian tiket movie yang akan tayang di bioskop kesayangan anda.
 </p>

## 🛠️ Installation Steps

1. Clone the repository

```bash
git clone https://github.com/guslizikri/tickitz-app.git
```

2. Install dependencies

```bash
npm install
# or
yarn install
```

3. Run the app

```bash
npm run dev
```

🌟 You are all set!
